# JGaming
> JGaming Engine follow Hazel
> 本游戏引擎开发追随@chernoYOUTUBE的教程还有《游戏引擎架构》这一本书。希望自己能保持周更！

- 入口点（是所有游戏引擎入口的地方，主要维持一个主循环）
- 应用层（构建真正的游戏引擎的应用程序，分层的）
- 窗口层
  - 处理输入
  - 事件处理
- 着色器（给模型上色等）
- 着色器的应用程序接口
- 日志系统（用于处理异常和bug之类的东西）
- 脚本语言（加入脚本语言接口就可以实现自定义的高级功能）
- 内存管理系统
- 组分实体管理系统（简单来说就是导入的资源管理的系统）
- 物理系统
- 文件输入输出系统，虚拟文件系统
- 构建系统
- etc
